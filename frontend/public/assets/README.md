# Dossier Assets

Placez vos fichiers téléchargeables dans ce dossier.

## Fichiers requis

### CV
- **Nom** : `cv.pdf`
- **Format** : PDF
- **Taille** : < 5 MB
- **Recommandation** : 1-2 pages maximum

## Autres fichiers possibles

Vous pouvez également ajouter :
- Lettres de recommandation
- Certificats (PDF)
- Portfolio détaillé (PDF)
- Présentation (PowerPoint/PDF)

## Note

Assurez-vous que le lien de téléchargement dans `index.html` correspond au nom exact de votre fichier :

```html
<a href="assets/cv.pdf" class="btn btn-primary" download>
```

Si votre CV a un autre nom, modifiez le lien ou renommez votre fichier.
