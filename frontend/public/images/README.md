# Dossier Images

Placez vos images dans ce dossier.

## Images requises

### Photo de profil
- **Nom** : `profile.jpg` (ou .png)
- **Dimensions recommandées** : 500x500px minimum (format carré)
- **Format** : JPG ou PNG
- **Taille** : < 500 KB

### Images de projets
- **Noms** : `project1.jpg`, `project2.jpg`, `project3.jpg`, etc.
- **Dimensions recommandées** : 800x600px ou ratio 4:3
- **Format** : JPG ou PNG
- **Taille** : < 300 KB par image

### Favicon
- **Nom** : `favicon.png`
- **Dimensions** : 32x32px ou 64x64px
- **Format** : PNG avec transparence

## Optimisation

Pour optimiser vos images :

1. **En ligne** :
   - https://tinypng.com/
   - https://squoosh.app/

2. **Conversion en WebP** (recommandé pour la performance) :
   - https://cloudconvert.com/jpg-to-webp

3. **Redimensionnement** :
   - https://www.iloveimg.com/resize-image

## Note

Si une image n'est pas trouvée, le site affichera automatiquement un placeholder.
